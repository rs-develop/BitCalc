# BitCalc

## What is BitCalc?
*BitCalc* is a basic console application for calculating the
- ASCI value,
- decimal value and the
- bitmap

of a max 16 byte hex value.
```
+----------------------------------------------+
| HEX-Value: 55                                |
+----------------------------------------------+
| ASCII:     U                                 |
+----------------------------------------------+
| DEC:       85                                |
+----------------------------------------------+
| Bytes:     1                                 |
+----------------------------------------------+
|  0000 0000  0000 0000  0000 0000  0101 0101  |
|  31                    15                 0  |
+----------------------------------------------+
```

## How to build
`1. cmake .`
`2. make`
